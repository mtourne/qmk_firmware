# QMK Keyboard Firmware for HHKB

## Deploy

build:
`make hhkb/ansi:mtourne`

deploy:
`make hkkb/ansi:mtourne:dfu`

# Regular HHKB

**plus**

* hardwired dvorak
* SpaceFn movements
* mouse keys
* in-keeb modifiers


# Extra Layers

## SpaceFn
    * qwer `ijkl` movement keys

## Mouse
    * `ijkl` mouse movements
    * activated by holding `f`(u) on the other side
    * locked with `d`(e)
